# Translation Conventions

## Key Naming
*   Use camelCase or kebab-case consistently (e.g., `traditionalGames`, `main-costume`).
*   Prefix keys with region IDs when possible for clarity.

## Cultural Context
*   Ensure that terms like "ethnographic" and specific names of costumes or traditions are translated with cultural sensitivity.
*   Preserve the original names of regions and subzones if applicable, providing translations where necessary for clarity.
