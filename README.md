# Ethnographic Interactive Map

Interactive map showcasing regional ethnographic traditions and cultural knowledge.

## Stack

- Next.js (App Router + TypeScript)
- CSS Modules
- Radix UI
- react-simple-maps
- Local JSON content + Zod validation

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Current MVP scope

- Interactive map with clickable regions
- Region content panel (games, costumes, traditions)
- 3 seeded regions: Maramures, Moldova, Oltenia
- Local static assets and data

## Deploy

1. Push `main` branch to GitHub.
2. Import repository in Vercel.
3. Keep default framework detection (Next.js).
4. Deploy.
